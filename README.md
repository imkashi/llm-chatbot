# llm-chatbot
Chatbot using LLM

1. install depndency:-
pip install openai streamlit

2. structure:-\n
llm-chatbot/
|--.streamlit/
  |---secrets.toml       # to put your OPENAI_API_KEY
├── app.py               # Main Streamlit app
└── requirements.txt     # Python dependencies

4. run:-
streamlit run app.py
